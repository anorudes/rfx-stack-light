# RFX Stack
based on [rfx-stack](https://github.com/foxhound87/rfx-stack)

#### Universal App featuring:
### React + Feathers + MobX

## Setup

##### ENV: Development

`npm install`

`npm run server:dev`

> Run the seeder in another terminal window while the api server is running.

##### ENV: Production

`npm install`

`npm run build:all`

`npm run server:prod`
